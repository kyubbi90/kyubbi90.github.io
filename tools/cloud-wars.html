Cloud Wars

Notes were taken while watching this youtube video.

  • Azure has become more popular over the years
  • Biggest difference between cloud and traditional design is horizontal scaling
  • Both scaling up and scaling down are important though people mostly talk about scaling up
  • The region the server is in and the availability zone are important based on users of sites.
  • Microsoft uses current products to segue into cloud products with ease.
  • All Azure features can be accessed through VS Code.
  • Node Js is usable with Azure.
  • Azure Active Dir makes it easy to update to cloud automatically.
  • Google custom builds everything even things on data servers
  • Google offers the most services dealing with big data
  • There are other cloud providers outside of these three such as IBM
  • Hybrid clouds can be done by using multiple clouds
  • You are not locked into a single cloud
  • Getting certified in 1 cloud and finding an employer that uses that cloud is a good way to start
  • Most places use the cloud or will use the cloud in the future.
