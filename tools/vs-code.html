Visual Studio Code for Web Developers

Overview of VS Code

The VS Code UI

  • Can manually open folder through VS code or drag folder to vs code
  • Ctrl-shift-g - quick open git view
  • Can install extension in editor
  • Command palette - shows all commands even ones from extensions
  • Zen mode - all UI disappears besides code

Basic Editing

  • Copying an entire line if no text is selected
  • Ctrl+] - Indent to the right
  • Alt+shift+f - Format
  • Ctri+I - select full line

The Integrated Terminal

  • Ctrl+` - Open integrated terminal
  • Can open multiple terminals
  • Ctrl + k - clears terminal

Integrated Source Control

  • Changes are shown in the git view
  • They can be unstaged

Customizing VS Code

Configuring preferences

  • Enter new settings on right side
  • Add property to left side with new value
  • Can set preferences for different languages

Setting keyboard shortcuts

  • Shortcuts are customizable like preferences
  • All shortcuts are customizable

Using extensions

  • Extensions downloadable from the editor or the marketplace website
  • Extensions give details of how to use

Essential extensions

  • Html snippets - gives shortcuts to auto complete code
  • Beautify - formats code to read easier

Working with snippets

  • Some snippets are built in to vs code
  • Snippets can be added in preferences or with extensions

Color and icon themes

  • Color themes can be downloaded from the marketplace
  • Icon themes are also available

Using the Editor

Files, folders, and projects

  • Sidebar can be hidden or shown with CTRL+B
  • Ctrl+tab - changes tabs

Side by side editing

  • Ctrl +\ - opens preview on right side

Intellisense

  • Ctrl+space opens intellisense manually
  • Parameter options are displayed

Finding and replacing

  • Ctrl + F -search
  • Can search all the files in a project

Advanced Editing

Editor features

  • Hot exit - remembers content of unsaved files
  • Auto save - available to activate at top bar
  • Format on save can be turned on in preferences

Web code editing

  • Ctrl + / - comments out selected code
  • Gives easier way to document code

Navigating code

  • Ctrl + g - go to line
  • Ctrl + shift + o - show elements to go to
