Front-end Frameworks

I decided to research AngularJS, I have never used it before but I've heard a lot about it. The first place I decided to check was the official AngularJS website. Angular specializes in extending HTML to become more dynamic. Angular works 100% on the client-side. I also watched this video by JsWiz. USA Today's website is an example of a website that uses AngularJS. Angular is loaded through the browser compared to other sites which are usually loaded through the server. Loading through the browser makes the site run much faster than server loaded sites. Angular 2 had a large change from Angular 1 including a code rewrite from top to bottom.
